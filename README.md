#Overview
This program takes in 3 integers and tests to see if they can make an Isoscles Triangle.

#Running the Program
To Compile and Run
	go run isosceles.go

To Test
	go test -bench=.

#Known Issues
isoscles_test.go cannot test if the correct error was returned only that an error was returned when it was
supposed to be.
